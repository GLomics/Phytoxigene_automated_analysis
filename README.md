# Phytoxigene Automated Analysis

SUMMARY OF METHOD

R-code and the accompanying directory/files are used to take the “raw” results from  Phytoxigene Total Cyanobacteria and Toxin Gene assays, perform QC, then provide final concentrations of each respective target. Both qPCR assays are processed following the same steps, with an ability to analysis samples run for Total Cyanobacteria and mcyE Toxin gene in unison.  Previous standards can be included into the analysis to allow for easy processing of projects that span multiple runs.
